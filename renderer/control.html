Walls

Each wall has its own size, panel layout and orientation. Select one to edit it below; assign walls to outputs in the Outputs panel. “Window” opens the wall in a virtual output for pre-vis.

Wall Setup

px = round(mm ÷ pitch). Spec-sheet pitch is often nominal — e.g. many “P2.9” 500 mm panels are really 168×168 (true pitch 2.976). If you know the panel’s real resolution, fine-tune the pitch until the pixels match.

Pattern

Rendered into the pattern itself — shows on outputs, preview and exports. The center label is the output's label (or the wall name if none is set).

Overlay Pulse

Sweeps over the active pattern — watch it cross panel seams to catch mapping errors.

Outputs

Physical displays appear automatically — graphics card outputs, or a playback card whose driver presents it as a display. Virtual outputs are windows that behave like a display of the given resolution, so you can map the whole show before hardware is connected. Press ESC on an output to close it.
